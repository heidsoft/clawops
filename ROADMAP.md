# ClawOps Roadmap - AI DevOps Agent Platform

> **产品定位**: 开源的 AI 驱动的 DevOps 自动化平台
> **Slogan**: "你的第二个 DevOps 工程师"

---

## 愿景

```
用户: "帮我部署一个 Node.js 应用到阿里云"
AI:   "我来帮你完成部署，请确认以下配置：
       - 地域: 华北 2 (北京)
       - 实例规格: 2核4G
       - 环境: 生产环境
       
       开始执行..."
       
       → 创建实例 → 配置安全组 → 安装 Docker → 部署应用
       
AI:   "✅ 部署完成！
       - 访问地址: http://101.200.XXX.XXX:8080
       - 部署耗时: 3分25秒"
```

## 技术架构

### Agent 执行流程

```
┌─────────────────────────────────────────────────────────────┐
│                      User Input                             │
│                  "部署 nginx 到阿里云"                      │
└─────────────────────┬───────────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────────┐
│                   Intent Recognition                         │
│              识别为: deploy_skill_request                   │
└─────────────────────┬───────────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────────┐
│                    Skill Matching                           │
│               匹配到: deploy skill (v1.0)                   │
└─────────────────────┬───────────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────────┐
│                   Parameter Extract                          │
│         提取参数: { app: nginx, platform: aliyun }        │
└─────────────────────┬───────────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────────┐
│                    Execution Loop                            │
│  ┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐  │
│  │ Create  │───▶│ Config  │───▶│ Deploy  │───▶│ Verify  │  │
│  │ Instance│    │ Security│    │ App     │    │ Health  │  │
│  └─────────┘    └─────────┘    └─────────┘    └─────────┘  │
└─────────────────────┬───────────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────────┐
│                    Response                                 │
│       "✅ 部署完成！访问地址: http://..."                    │
└─────────────────────────────────────────────────────────────┘
```

### Skill 扩展机制

```yaml
# Skill 定义示例
name: deploy
version: 1.0.0
description: 自动化部署技能
platforms:
  - aliyun-ecs
  - aws-ec2
  - docker
  - kubernetes

parameters:
  - name: app
    type: string
    required: true
    description: 应用名称
  - name: platform
    type: enum
    options: [aliyun, aws, docker, k8s]
    default: docker

steps:
  - name: create-instance
    action: cloud.createInstance
  - name: config-security
    action: cloud.configureSecurityGroup
  - name: deploy-app
    action: app.deploy

output:
  - access_url
  - instance_id
  - deployment_time
```

## 贡献指南

### 如何贡献 Skill

1. Fork 仓库
2. 在 `harness/skills/contrib/` 创建 Skill
3. 编写 SKILL.md
4. 提交 PR
5. 审核通过后合并

### 开发环境

```bash
# 克隆
git clone https://github.com/heidsoft/clawops.git
cd clawops

# 启动后端
cd backend && go run cmd/server/main.go

# 启动前端
cd frontend && npm install && npm run dev

# 启动 LLM (可选，本地模式)
ollama serve
```
